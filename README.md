# cvcstats

A simple webapp for viewing player statistics for the Cops vs. Crims minigame on Hypixel Network, powered by Hypixel's public Developer API and the official Mojang api. 

## About
This app uses Tanstack Start and Convex (convex backend is WIP), with TailwindCSS and ShadCN for styling.
